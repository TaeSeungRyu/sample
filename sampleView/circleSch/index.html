1. 원형 내부 스케줄은 더블클릭시 사라집니다.
2. 스케줄 추가는 우측 매뉴에서 아이템을 클릭하고 원형 스케줄에 마우스를 누른 채 움직 입니다.
3. 반응형을 지원하나 좌우측 스크롤바가 생긴 경우는 아직 지원되지 않습니다.
4. 여기서 조작은 할 수 없으나 제공되는 옵션입니다.

5. 아래는 콘솔에서 사용 가능한 함수 입니다.
rts06069.getTimePureData(); //가공되지 않는 순수 시간과 관련된 값을 반환 합니다.
rts06069.getTimeData(); //시간단위로 값을 반환 합니다.
rts06069.getSummeryData(); //스케줄을 그룹핑하여 시간단위로 반환 합니다.
rts06069.exportImage('사진.png'); //이미지를 출력합니다. 이름은 옵셔널 입니다.
rts06069.getImageData(function(blobData){ console.log(blobData) }); //이미지를 블랍으로 받습니다.
rts06069.clearAll() //스케줄 내부를 초기화 합니다.
rts06069.resizeOnOff = true //반응형 여부를 설정 합니다.
